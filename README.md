# Crypto Tracker

- A website to track the crypto currency marker with data for 100 coins.

# Built with:
- React.js
- CSS
- JavaScript
- CoinGecko API
- Axios

# Features
- Market cap data
- Volume of coin
- Price of coin
- Search function for coins
- Percentage of how much a coin is up or down in the market
